Описание JS-библиотеки Bitrix Framework. 

Основным классом JS-библиотеки Битрикс является класс BX. Также к услугам разработчика представлено достаточно большое количество расширений.

Все файлы - скрипты и стили библиотеки находятся в папке 

/bitrix/js/main/core/ 

Файл core.js - это ядро самой библиотеки. Ядро определяет глобальный объект BX, который содержит статические методы для работы с событиями, DOM и др. По сути это аналог  "$" в jquery. С него начинаются все действия и манипуляции с объектами страницы. 

 

Из папки /bitrix/js/main/core/ в большинстве нужны следующие файлы: 

  • core_ajax.js - ajax функции; 
  • core_date.js - функции для работы с датами и календарями; 
  • core_popup.js - функции для создания всплывающих окон; 
  • core_window.js - для создания диалоговых окон таких, как при редактировании контента в публичной части. С помощью этой библиотеки удобно загружать во всплывающие окна админские страницы и части админки. 

 

Использование библиотеки в своем коде

Чтобы ваш код смог запуститься только когда DOM-структура будет полностью загружена и сформирована, необходимо использовать внутри метода BX.ready():

BX.ready( function() {
    // ваш код
});

Для работы с BX нужно усвоить два момента:

  • Все методы библиотеки работают с элементами DOM-структуры, в отличии, например, от jQuery, где возвращаются специализированные объекты, с которыми можно проводить различные манипуляции. В BX необходимо сначала получить элемент или массив элементов, и производить действия с каждым по отдельности. Ссылку на отдельный элемент можно получить методом BX(‘атрибут id DOM-элемента’);
  • Большинство методов используют два параметра - первым идет элемент, вторым - данные о действиях, применяемых с элементу (например, BX.setOpacity). Оставшаяся часть методов работает в формате BX.метод(элемент) (к примеру, BX.focus).

Для примера, попробуем удалить все изображения в боковой панели сайта:

BX.ready(function(){
    // выбираем все изображения в массив
    var images = BX.findChild(BX('side-panel'), {tag: 'img'}, true, true);
    // перебираем массив и к каждому элементу применяем метод удаления
    images.forEach(function(element){
        BX(element).remove();
    }); 
});
