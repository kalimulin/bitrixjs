BX.cleanNode

DOMNode|null 
BX.cleanNode(
 DOMNode node[,
 bool bSuicide]
);

Функция очищает узел DOM-структуры от всех дочерних элементов. Если вторым параметром передать true, сам узел также будет удален.


Пример

<script>
BX.ready(function() {
	BX.bind(BX('buttonDivId'), 'click', function() {
		//Очищает узел DOM-структуры от всех дочерних элементов
		BX.cleanNode(BX("spanid"));
	});
	BX.bind(BX('buttonDivId_2'), 'click', function() {
		//Очищает узел DOM-структуры от всех дочерних элементов и сам узел также будет удален
		BX.cleanNode(this, true);
	});
});
</script>

<div id="buttonDivId" class="buttonDiv"> cleanNode <span id="spanid">clean</span> </div>
<div id="buttonDivId_2" class="buttonDiv"> cleanNode <span id="spanid_2">remove</span> </div>
