BX.width/BX.height

Возвращает (или задает) ширину, высоту элемента DOM. Первым параметром передаётся элемент DOM. Если не передавать второй параметр (целое число), то вернется значение ширины или высоты. Если передать - значение задается. 

 

Примеры использования

BX.height(BX('adm-title'));

BX.height(BX('adm-title'), 1000);

 

<script>
BX.ready(function() {
	BX.bind(BX('buttonDivId'), 'click', function() {
		//задает) ширину
		BX.width(this, 300); //BX.width(BX('buttonDivId'), 300);
		this.innerHTML = "now width " + BX.width(this) + "px";
	});
});
</script>

<div id="buttonDivId" class="buttonDiv">set width 300</div>
