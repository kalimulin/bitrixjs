BX.addClass

DOMNode 
BX.addClass(
 DOMNode node,
 string className
);

Функция добавляет узлу node CSS-класс className, если он такового не имеет.

 

BX.removeClass

DOMNode 

BX.removeClass(

 DOMNode node,
 string className

);

Функция удаляет у узла node CSS-класс className.

 

BX.toggleClass

DOMNode 
BX.toggleClass(
 DOMNode node, 
 string|array className
);

Функция переключает наличие/отсутствие CSS-класса className у узла node или производит ротацию CSS-классов, если className – массив.

 

BX.hasClass

boolean
BX.hasClass(
 DOMNode node,
 string className
);

Функция проверяет, есть ли уже у узла node CSS-класс className.

 

Примеры

BX.ready(function(){
  BX.addClass(BX("elemId"), 'className');

  BX.removeClass(BX("elemId"), 'className');

  BX.bind(BX("test"), "click", function() {
      BX.toggleClass(BX("test"), ["arrowUp", "arrowDown"]);
  });
  
  BX.hasClass(BX("elemId"), 'className');
})
