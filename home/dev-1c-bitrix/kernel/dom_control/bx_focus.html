BX.focus

BX.focus(
 DOMNode node
);

Устанавливает фокус на поле ввода.

 

Параметры функции

 

Примеры

<script>
 BX.ready(function(){
    BX.bind(BX('togglediv'), 'click', function() {
        BX.focus(BX('t3'));    
    });    
 });
</script>

<div id="togglediv" class="arrowUp">focus on t3</div>

<input id="t2" type="text" maxlength="20" value="t2">
