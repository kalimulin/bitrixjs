BX.style

string|DOMNode 
BX.style(
 DOMNode node,
 string property[,
 string value]
);

Функция позволяет получить текущее значение стиля property узла node или установить его в значение value. Свойство property указывается в CSS-нотации (например, 'margin-right')

Примеры

BX.style(document.body, 'margin', '100px 200px');
alert(BX.style(document.body, 'margin-top')); //100px

Установка прозрачности

<script>
BX.ready(function() {
	BX.bind(BX('buttonDivId'), 'click', function() {
		//Метод устанавливает уровень прозрачности элемента веб-страницы
		BX.style(this, "opacity", "0.3"); //BX.setOpacity(BX('buttonDivId'), "opacity", "0.3");
	});
});
</script>

<div id="buttonDivId" class="buttonDiv">style opacity to 30%</div>
