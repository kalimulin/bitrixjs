BX.scrollToNode

BX.scrollToNode(
 DOMNode node
);

Прокручивает окно к позиции указанного элемента в DOM-структуре.

 

Пример

<script>
 BX.ready(function(){
    BX.bind(BX('togglediv'), 'click', function() {
        BX.scrollToNode(BX('footer'));
    });    
 });
</script>

<div id="togglediv" class="arrowUp">scrollToNode footer</div>
