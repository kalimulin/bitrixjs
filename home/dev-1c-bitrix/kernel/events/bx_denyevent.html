void 
BX.denyEvent(
 DOMNode element,
 string event
); 

void 
BX.allowEvent(
 DOMNode element,
 string event
);

Два метода, позволяющих временно отключить и включить обработку конкретного события в конкретном элементе. Пока что отключаются только обработчики, назначенные через element['on' + event].
